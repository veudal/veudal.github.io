HOW TO UPLOAD
============

Files can be uploaded here under the condition that the access key is valid.

Note: The maximum allowed file size is 1000 MiB.

TERMS OF SERVICE
----------------

veudal.github.io is NOT a platform for:
    - piracy
    - pornography
    - extremist material of any kind
    - malware / botnet
    - backups
    - doxxing, database dumps containing personal information
    - anything illegal under German law

Uploads found to be in violation of these rules will be removed,
and the originating IP address blocked from further uploads.


FILE RETENTION PERIOD
---------------------

Files are kept for a minimum of 7, and a maximum of 180 Days.

How long a file is kept depends on its size. Larger files are deleted earlier 
than small ones. This relation is non-linear and skewed in favour of small 
files.

The exact formula for determining the maximum age for a file is:

MIN_AGE + (-MAX_AGE + MIN_AGE) * (FILE_SIZE / MAX_SIZE - 1)^5
